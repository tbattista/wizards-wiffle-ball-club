Game Setup

  • This is a fun ALL AGES game. (must be atleast 6)
  • 5 Inning Game
  • 3 outs per inning
  • No gloves
  • Bats and balls provided
  • All bats must be standard yellow plastic wiffle ball bats (taped/customized is fine)
